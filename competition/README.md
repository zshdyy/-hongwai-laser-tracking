# 电赛题目与解析 | Electronics Competition

本模块整理全国大学生电子设计竞赛（NUEDC / 电赛）中与**视觉识别**和**模拟电路**相关的经典题目，提供题目分析框架和参考设计思路。

> ⚠️ 参考设计仅供学习参考，请遵守竞赛规则，所有作品须由参赛队独立完成。

---

## 视觉类题目常见考点

```
视觉类题目通用方案框架
├── 图像采集
│   └── 摄像头（USB / CSI）+ 树莓派 / OpenMV
├── 目标识别
│   ├── 颜色识别（HSV 分割）
│   ├── 形状识别（轮廓 + 多边形拟合）
│   └── 二维码 / 条形码（pyzbar / ZXing）
├── 坐标计算
│   └── 像素坐标 → 实际坐标（透视变换）
└── 执行机构控制
    ├── 舵机（角度控制）
    ├── 步进电机（精确位移）
    └── 直流电机（速度控制 + PID）
```

## 模拟类题目常见考点

| 考点 | 关键技术 |
|------|---------|
| 信号放大 | 运放增益设计、带宽计算 |

| 波形发生 | DDS（AD9833）、NCO |
| 电源管理 | 高效率 DC-DC 设计 |
| 测量仪器 | 频率计、数字电压表 |

## 备赛建议

1. **提前练习**：至少完整做过3道往届题目（从理论到实物）
2. **组内分工**：模拟方向 + 数字/单片机方向 + 综合协调
3. **器件储备**：提前采购常用芯片和模块，避免赛中等货
4. **文档规范**：设计报告是评分重要依据，平时养成记录习惯
5. **联调测试**：赛前1个月进行软硬件联合调试，发现隐患
