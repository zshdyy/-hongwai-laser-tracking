# 振荡电路实验 | Oscillator Circuits

## 实验目标

- 理解正弦波振荡的相位条件与幅度条件
- 搭建文氏电桥振荡器（Wien Bridge）和 NE555 方波振荡器

---

## 常见振荡器类型

| 类型 | 频率范围 | 波形 | 特点 |
|------|---------|------|------|
| RC 文氏电桥 | 1Hz ~ 1MHz | 正弦波 | 频率可调，失真小 |
| LC 振荡器 | 1MHz ~ GHz | 正弦波 | 高频适用 |
| NE555 多谐振荡 | 1Hz ~ 500kHz | 方波 | 简单易用 |
| 晶体振荡器 | 固定频率 | 正弦/方波 | 精度极高 |

---

## NE555 方波振荡器（快速入门）

```
        +Vcc
         │
        [R1]
         │
    ─────┤8,4  NE555  3├──── 方波输出
    │    │              │
   [C1] [R2]   2,6      │
    │    │─── ─┘        │
    └────┘    5├──[C2]──GND
              1
              │
             GND
```

频率：`f ≈ 1.44 / ((R1 + 2·R2) · C1)`

占空比：`D = (R1 + R2) / (R1 + 2·R2)`

---

## 实验步骤

1. 选取 R1 = 1kΩ，R2 = 10kΩ，C1 = 10nF，计算预期频率
2. 搭建电路，用示波器测量输出频率与占空比
3. 调整 R2 值，观察频率变化规律
