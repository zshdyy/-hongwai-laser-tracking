# 滤波器设计实验 | Filter Design

## 实验目标

- 设计并验证低通、高通、带通滤波器
- 掌握截止频率计算与频率响应测量方法

---

## 基本滤波器类型

| 类型 | 功能 | 典型应用 |
|------|------|---------|
| 低通滤波器 (LPF) | 保留低频，滤除高频 | 音频平滑、ADC 抗混叠 |
| 高通滤波器 (HPF) | 保留高频，滤除低频 | 去除直流偏置、音频高音补偿 |
| 带通滤波器 (BPF) | 保留特定频段 | 通信系统、信号选频 |
| 带阻滤波器 (BEF) | 滤除特定频段 | 工频 50Hz 陷波 |

---

## 一阶无源低通滤波器

```
Vin ──┤R├──┬── Vout
           │
          [C]
           │
          GND
```

截止频率：`fc = 1 / (2π·R·C)`

示例：R = 1kΩ，C = 0.1μF → `fc ≈ 1.59kHz`

---

## 二阶有源低通滤波器（Sallen-Key）

特点：
- Q 值可调，实现 Butterworth / Chebyshev 特性
- 使用运放作为有源元件，输出阻抗低

---

## 实验步骤

1. 计算 RC 元件值，使 fc = 1kHz
2. 用信号发生器扫频（100Hz ~ 100kHz），记录输出幅度
3. 绘制幅频特性曲线（Bode 图）
4. 与仿真结果对比，分析元件误差的影响
