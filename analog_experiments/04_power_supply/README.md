# 电源电路设计实验 | Power Supply Design

## 实验目标

- 理解线性稳压与开关稳压的区别与适用场景
- 设计并测试 LM7805 线性稳压电路与 Buck 降压电路

---

## 线性稳压器（LM7805）

```
Vin(7~35V) ──[LM7805]── 5V 输出
                │
               GND
```

- 优点：纹波小、设计简单
- 缺点：效率低（多余功率以热能耗散）

电路要点：
- 输入端加 0.1μF 去耦电容
- 输出端加 0.33μF 稳定电容
- 输入输出压差 ≥ 2V（压差损耗）

---

## 开关电源（Buck 降压）

| 参数 | 典型值 |
|------|--------|
| 输入电压 | 7 ~ 24V |
| 输出电压 | 3.3V / 5V（可调） |
| 开关频率 | 50kHz ~ 500kHz |
| 效率 | 85% ~ 95% |

常用芯片：`LM2596`、`XL4016`、`MP1584`

---

## 效率对比测试

1. 分别搭建 LM7805 与 LM2596 电路，输入 12V，输出 5V
2. 在输出端接 100Ω 负载（50mA 输出）
3. 分别测量输入电流，计算效率 `η = Pout / Pin × 100%`
4. 比较两种方案的优缺点，记录发热情况
