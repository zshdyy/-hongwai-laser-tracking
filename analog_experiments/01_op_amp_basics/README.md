# 运算放大器基础实验 | Op-Amp Basics

## 实验目标

- 理解运算放大器的"虚短"（V⁺ ≈ V⁻）与"虚断"（I⁺ ≈ I⁻ ≈ 0）概念
- 搭建并测试同相放大器、反相放大器、加法器、差分放大器

---

## 电路原理

### 反相放大器

```
       R1            Rf
Vin ──┤├──┬──┤├──┬── Vout
         │         │
         └── U1- ──┘
              │
             U1+
              │
             GND
```

增益：`Av = -Rf / R1`

### 同相放大器

增益：`Av = 1 + Rf / R1`

## 实验步骤

1. 按原理图在面包板上搭建反相放大器（R1 = 10kΩ，Rf = 100kΩ）
2. 输入 1kHz、100mVpp 正弦波
3. 用示波器同时观测输入与输出，验证增益和相位关系
4. 改变 Rf 阻值，观察增益变化

## 常见问题排查

| 现象 | 可能原因 | 解决方法 |
|------|---------|---------|
| 输出波形削顶 | 输入信号过大导致饱和 | 减小输入幅度 |
| 输出为 0 | 电源未接通 / 芯片方向错误 | 检查供电与引脚方向 |
| 增益不符 | 电阻读数错误 | 用万用表测量实际阻值 |
